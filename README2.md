Smart Object Detecting Stick

📌 Overview

The Smart Object Detecting Stick is an assistive device designed to help visually impaired individuals detect obstacles in their path. It uses sensors and a microcontroller to identify objects and alert the user in real-time.

---

🚀 Features

- Obstacle detection using ultrasonic sensor
- Real-time alerts (buzzer/vibration)
- Lightweight and portable design
- Low power consumption
- Easy to use

---

🛠️ Technologies Used

- Arduino
- Ultrasonic Sensor 
- Buzzer 
- Arduino IDE

---

🔌 Components Required

- Arduino Board
- Ultrasonic Sensor
- Buzzer
- Battery
- Connecting Wires
- Stick (physical support)

---

⚙️ Working Principle

The ultrasonic sensor continuously measures the distance between the stick and nearby objects.
If an object is detected within a predefined range, the system alerts the user through sound or vibration.
